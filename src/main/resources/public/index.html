PoC StateMachine

Esta é uma aplicação para testes do Spring StateMachine.

A aplicação consiste em uma lista de tarefas. Quando uma nova tarefa é cadastrada ela fica como "NOT_STARTED". Ao iniciar a tarefa evolui para "WORKING" e é chamado um serviço que pode demorar até 1 minuto para responder. Caso o serviço responda com sucesso em até 30s a tarefa evolui para "DONE" e é concluída. Quando o serviço demora mais de 30 segundo para responder a tarefa evolui para "ERROR" e aguarda 30s para repetir a atividade, evoluindo o estado de volta para "WORKING". Após 3 tentativas mal sucedidas de completar a tarefa, seu estado passa para "PENDING" quando o usuário pode tentar manualmente.
